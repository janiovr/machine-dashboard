# CNC Machine Dashboard

Interactive dashboard simulating a CNC / 3D printer control system.

## 🚀 Features
- Real-time axis coordinates (DRO)
- Spindle control (RPM)
- Feed rate adjustment
- Jog control interface
- G-code viewer
- System logs
- Emergency stop simulation

## 🛠️ Technologies
- HTML
- CSS
- JavaScript

## 💡 About
This project simulates an industrial machine interface, focusing on usability, real-time feedback, and system control logic.
